# Advanced-Sales-Analysis-Dashboard
# 📊 Enterprise Sales Analytics Dashboard (Power BI)

🎯 Project Overview
A comprehensive Power BI business intelligence solution designed to analyze and visualize sales performance across multiple business dimensions.

This project delivers a full analytical experience tailored for both executive leadership (C-Level) and operational sales teams, enabling data-driven decision-making through interactive dashboards and advanced analytics.


🌟 Key Features

🧑‍💼 Executive Dashboard (C-Level)
- High-level KPIs (Revenue, Profit, Margin %, Orders)
- KPI tracking (Actual vs Target, LY, YTD)
- KPI performance summary (Achieved vs Under Target)
- Strategic insights with clean and professional layout


📊 Sales Operational Dashboard
- Detailed sales performance tracking
- Monthly trends (Ordered Amount vs Delivered Amount)
- Category & Subcategory performance
- Country-level sales analysis


📈 Advanced Analytical Pages

🔹 Delivery Performance Analysis
- Monthly trend: Ordered vs Delivered Amount
- Dynamic tooltip comparing last 3 years

🔹 KPI & Target Analysis
- KPI vs LY
- KPI vs LY YTD %
- Actual vs Target tracking

🔹 Top Performers Analysis
- Dynamic Top 5 Subcategories
- Automatically updated based on slicers

🔹 Geographic Analysis
- Country-level sales distribution
- Drill-through for historical analysis (independent of slicers)

🔹 Sales Contribution Analysis
- Country contribution % of continent total
- Not affected by country slicers

🔹 Delivery Forecasting
- Orders scheduled within:
  - 1 Week
  - 15 Days
  - 1 Month
  - 6 Months

🔹 Promotions Analysis
- Active promotions across time (including open-ended promotions)

🔹 KPI Performance Monitoring
- Number of KPIs achieving target
- Number of KPIs under target
- Tooltip breakdown of KPI details

🔹 Marketing Analytics
- YTD vs LY comparison
- Dynamic time selection:
  - Last 15 / 30 / 45 Days
- Manual period comparison (Period 1 vs Period 2)


🛠️ Technical Implementation

📐 Data Modeling
- Star Schema design
- Fact & Dimension tables
- Data model optimization and relationship corrections


⚙️ DAX Calculations
- Time Intelligence:
  - YTD
  - LY
  - LY YTD
- KPI Calculations:
  - Revenue, Profit, Margin %
  - Target vs Actual
- Advanced Metrics:
  - Growth %
  - Dynamic comparisons


🔐 Row Level Security (RLS)
- User-based access control:
  - User A → Full access to TV & Video category
  - User B → Access to specific subcategory


🎨 Dashboard Design
- Clean and modern UI
- Consistent color palette
- Proper alignment and spacing
- Use of icons and conditional formatting


📊 Business Insights
- Identified top-performing categories and subcategories
- Analyzed delivery performance gaps
- Evaluated KPI achievement vs targets
- Provided geographic sales distribution insights
- Assessed promotional effectiveness
